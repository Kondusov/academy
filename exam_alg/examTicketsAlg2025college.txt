Билеты для экзамена по предмету "Основы алгоритмизации и программирования":

Билет 1.
Задача. Реализуйте пузырьковую сортировку (Сортировка простыми обменами) для массива целых чисел. 
Например, для массива [7,5,4,2,3,1,8,6] результат должен быть [1,2,3,4,5,6,7,8].
Вопрос1. Жадные алгоритмы.
Вопрос2. Структуры данных - стек и очередь.

Билет 2.
Задача. Напишите функцию, которая выполняет бинарный поиск в массиве положительных чисел.
Функция возвращает индекс искомого элемента, в случае если не находит - возвращает -1.
Вопрос1. Структура данных - граф.
Вопрос2. Связанные списки, двусвязные списки.

Билет 3.
Задача. Напишите функцию, которая проверяет сбалансированна строка или нет,
Например, если скобка в строке была открыта и далее закрыта то должны вернуть true, а иначе возвращаем — false.
Вопрос1. Структура данных - множество.
Вопрос2. Приближенные алгоритмы.

Билет 4.
Задача. Реализуйте структуру данных - стек.
(для примера добавьте в стек несколько элементов, и удалите один. Выведите содержимое стека.)
Реализуйте структуру данных - очередь.
(для примера добавьте в очередь несколько элементов, и удалите один.  Выведите содержимое Очереди.)
Вопрос1. Как оценить сложность алгоритма
Вопрос2. Массивы

Билет 5.
Задача. Реализуйте сортировку выбором для массива строк. Например,
для входного массива ["banana", "apple", "kiwi"] результат должен быть ["kiwi", "apple", "banana"].
(отсортировали по колличеству символов)
Вопрос1. Поиск в ширину
Вопрос2. Алгоритм K ближайших соседей.

Билет 6.
Задача. Реализуйте сортировку выбором для массива строк. Например,
для входного массива ["banana", "apple", "kiwi"] результат должен быть ["apple", "banana", "kiwi"].
(отсортировали по алфавиту)
Вопрос1. Хеш таблицы.
Вопрос2. Структура данных - граф.

Билет 7.

Например, для массива [1, 2, 3] результат должен быть 6.
Вопрос1. Алгоритм Дейкстра.
Вопрос2. Алгоритм быстрой сортировки.

Билет 8.
Задача. Реализуйте жадный алгоритм для решения задачи о рюкзаке
(она же задача размена монетами, купюры).
Вопрос1. Сравните алгоритм поиска в ширину и алгоритм Дейкстры.
Вопрос2. Массивы.

Вопрос2. Структуры данных - стек и очередь.

Билет 14.
Задача. Реализуйте сортировку простыми обменами (пузырьковая сортировка) для массива целых чисел.

Билет 15.
Задача. Реализуйте сортировку для массива строк. Например, для массива ['banana', 'apple', 'one', 'kiwi']

Вопрос1. Массивы (всё о массивах).
Вопрос2. Циклы (for, foreach, while, do while).
